Paper articles are a way to visually structure your pages content with color

Articles can have contrast or use the default background color

Articles can have every background-color specified in the Material Color Styles

To use those colors you have to install paper-styles and import the color.html element into your dom.

You have to import it yourself so the element is still fast, even if you are not using the material colors.

If the background color is too light, the text-color automatically switches to a dark font

You can also use your own colors and provide it via the hex value

The element calculates if the background color is to bright or dark and will adjust the text color automatically

To use the element with your own custom CSS properties defined in your dom, you provide the property name as a color value and the element will auto detect its type
